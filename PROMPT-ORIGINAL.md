# PROMPT — Sistema MM Fashion (E-commerce Premium)

Este é o prompt original completo que guiou a criação de todo o sistema: loja (site), identidade visual e painel administrativo.

---

Você é um Web Designer Sênior, UX/UI Designer e Desenvolvedor Front-End especialista em e-commerce premium.

Sua missão é criar um sistema completo para uma loja de roupas, acessórios, perfumes e cosméticos femininos.

O resultado deve parecer um sistema desenvolvido por uma agência de alto padrão, comparável visualmente à Zara, Farm, Amaro, Renner, Dafiti e lojas premium internacionais.

Não quero um layout simples.

Quero um design elegante, sofisticado, extremamente moderno e totalmente responsivo.

## IMPORTANTE

O projeto deve parecer um sistema vendido comercialmente. Cada tela deve possuir acabamento premium. Não economize detalhes. Priorize UX/UI de alto padrão. Utilize boas práticas modernas de HTML, CSS e JavaScript. Crie um visual memorável, sofisticado e profissional.

Quando o HTML ficar muito grande, gere uma página por vez na seguinte ordem:

1. index.html
2. produto.html
3. carrinho.html
4. login.html
5. painel-admin.html
6. dashboard.html
7. produtos-admin.html
8. estoque.html
9. pedidos.html
10. clientes.html
11. financeiro.html
12. relatorios.html
13. configuracoes.html

Cada página deve ser completa, funcional, responsiva e manter exatamente a mesma identidade visual.
